# Flickopedia 🎬

Flickopedia is a movie information website where users can search for movies and view detailed information such as the synopsis, cast, release date, and ratings. Built with React and styled using Tailwind CSS, Flickopedia leverages Vite for a fast development experience and optimized builds.

## Features

- **Search Functionality**: Quickly search for movies by title.
- **Movie Details**: View comprehensive details about each movie, including plot, cast, ratings, and more.
- **Responsive Design**: Optimized for both desktop and mobile devices for a seamless user experience.

## Tech Stack

- **Frontend**: React
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **API Integration**: Fetches movie data from an external API
- **Version Control**: Git & GitHub for repository management

## Usage

- Use the search bar to find movies by title.
- Click on a movie to view detailed information including its synopsis, cast, and ratings.
